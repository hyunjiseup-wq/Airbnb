# 서울 Airbnb 수요·기회지역·입지 추천 분석

> 서울 지역 Airbnb 데이터를 기반으로  
> **게스트 수요 분석 → 기회지역 도출 → 최종 입지 추천**의 흐름으로  
> 호스트 관점의 의사결정을 지원하는 데이터 분석 프로젝트입니다.

- **프로젝트 형태**: 4인 팀 프로젝트
- **분석 관점**: 수요 분석 + 공간 분석 + 입지 추천
- **주요 도구**: Python, Pandas, GeoPandas, QGIS, Tableau
- **인터랙티브 대시보드**: [Tableau Public에서 보기](https://public.tableau.com/views/_17739846998650/sheet0?:language=ko-KR&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

---

## 프로젝트 개요

이 프로젝트는 서울 Airbnb 시장을 대상으로  
단순 숙소 가격 비교가 아니라 **수요, 상권, 경쟁도, 성과, 운영 지표**를 함께 분석하여  
어떤 지역이 유망한지, 그리고 어떤 입지가 실제로 기회지역이 될 수 있는지를 도출하는 것을 목표로 했습니다.

분석은 다음 3단계로 구성했습니다.

1. **게스트 수요 분석**
   - 서울 전역의 방문 흐름과 체류 특성을 파악
2. **기회지역 분석**
   - 자치구 단위 성과·공급·경쟁 구조 비교
3. **입지 추천**
   - 행정동 단위 최종 후보 추천 및 전략 제안

---

## 문제 정의

서울 Airbnb 시장에서 중요한 질문은 다음과 같습니다.

- 서울 내에서 실제 수요가 집중되는 지역은 어디인가?
- 공급이 많아도 성과가 유지되는 지역은 어디인가?
- 경쟁 대비 성과가 좋은 기회지역은 어디인가?
- 최종적으로 어떤 행정동이 신규 호스트에게 유리한가?
- 지역 선택 이후 어떤 운영 전략이 필요한가?

본 프로젝트는 이 질문에 답하기 위해  
**수요 분석 → 기회지역 도출 → 입지 추천**의 구조로 설계되었습니다.

## 대시보드 구성

## 1. 서울 에어비앤비 게스트 수요 분석

서울 전체의 Airbnb 수요 흐름을 파악하기 위한 메인 대시보드입니다.  
방문 규모, 숙박 특성, 검색 카테고리, 주요 방문 상권 등을 한 화면에서 탐색할 수 있도록 구성했습니다.

### 주요 내용
- 전체 방문 규모 및 평균 숙박일 요약
- 검색 상위 카테고리 및 주요 플레이스
- 자치구별 방문자수/숙박일수 차이
- 서울 전역 수요 및 상권 분포 지도
- 외국인·트렌드 관련 시계열 확인
- 수요 해석을 위한 인사이트 패널

![게스트 수요 분석](./images/dashboard_01_demand.png)

### 대시보드 의미
이 대시보드는 서울 전체 수요 구조를 먼저 파악하고,  
어떤 지역이 단순 유동이 아니라 **체류 가능성이 높은 수요를 가지는지** 탐색하는 시작점 역할을 합니다.

---

## 2. 서울 에어비앤비 기회지역 분석

자치구 단위에서 성과와 경쟁 구조를 비교하여  
어느 지역이 상대적으로 유망한지 확인하는 대시보드입니다.

### 주요 내용
- 선택 자치구 기준 KPI 요약
- 성과지표 / 공급지표 / 선호도지표 / 문화관광지표 비교
- 자치구별 Top10 기회지역 표
- 기회지역 지도
- 자치구 분포 및 행정동 기회지역 분포 시각화

![기회지역 분석](./images/dashboard_02_opportunity.png)

### 대시보드 의미
이 단계에서는 서울 전체 수요 분석 결과를 바탕으로  
자치구별 성과와 경쟁 구조를 비교해 **기회지역 후보군을 좁히는 역할**을 합니다.

---

## 3. 서울 에어비앤비 입지 추천

기회지역 분석 결과를 바탕으로  
행정동 단위의 최종 추천 입지를 제안하는 대시보드입니다.

### 주요 내용
- 추천 행정동 및 숙소 수
- 교통편의 점수 / 관광매력도 점수 / 기숙소당 점포밀도
- 기회지역 Top10, 교통/관광/점포 관련 비교 차트
- 추천 행정동 목록
- 추천 행정동 위치 지도
- 추천 입지 Top3 및 추가 운영 전략 제안

![입지 추천](./images/dashboard_03_recommendation.png)

### 대시보드 의미
이 대시보드는 분석의 최종 단계로,  
실제 호스트 또는 의사결정자가 **어디를 우선 검토해야 하는지**를 구체적으로 보여주는 결과 화면입니다.

---

## 핵심 인사이트

- 서울 내 Airbnb 수요는 지역별로 균등하지 않으며 특정 자치구와 상권에 집중되는 경향이 나타났습니다.
- 수요가 많다고 해서 항상 기회지역은 아니며, 공급과 경쟁 구조를 함께 봐야 실제 기회가 드러났습니다.
- 일부 자치구는 경쟁이 높아도 성과가 유지되었고, 일부 지역은 경쟁 대비 성과가 높아 기회지역 후보로 해석할 수 있었습니다.
- 최종적으로는 자치구 수준 분석만으로는 부족하며, 행정동 단위까지 내려가야 실제 입지 추천이 가능했습니다.
