# Airbnb
서울시 에어비앤비 데이터를 기반으로 호스트에게 유리한 지역을 찾고, 지역별 운영 전략을 제안한 공간 기반 데이터 분석 프로젝트
# 주요 기여
• 프로젝트 목적과 분석 방향 정의 및 역할 분담 리딩

• 행정동·자치구 단위 공간 데이터 전처리와 EDA 수행

• 성과·운영·경쟁 지표 분석(RevPAR, ADR, OCC, 리스팅 밀집도 등)

• QGIS 기반 공간 분석, 지도 시각화, Tableau Opportunity Map 구축

• 행정동 단위 Opportunity Score 산출 및 최종 추천 지역 도출

# 핵심 성과
• 서울시 25개 자치구, 425개 행정동, 20,000개 활성 리스팅 분석
• 행정동 단위 Opportunity Score 산출
• 최종 4개 유망 행정동 추천
• 12개 핵심 운영 전략 제안
# 트러블 슈팅
• 프로젝트 초기 방향성: 단순 Airbnb 분석에서 끝나지 않도록 프로젝트 목적을 "호스트에게 유리한 지역을 찾고, 지역별 운영 전략까지 제안"으로 재정의하여 전략 제안형 프로젝트로 전환
• 좌표계 불일치: QGIS 재투영(Reproject) 기능으로 QGIS·GeoPandas·Tableau 간 좌표계를 통일해 위치 정보 매칭 오류 해결
